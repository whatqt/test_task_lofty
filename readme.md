# Weather CLI Tool

Скрипт для получения текущей погоды в указанном городе.

## Установка

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/whatqt/test_task_lofty
   ```

2. Переход в репозиторий:
    ```bash
    cd test_task_lofty
    ```

3. Установка зависимостей из файла:
    ```bash
    pip install -r requirements.txt
    ```

4. Настройте API ключ:

    1. Получите ключ на [OpenWeatherMap](https://openweathermap.org/).

    2. Переименуйте ```.env.example``` в ```.env```.

    3. Замените значение на свой API ключ.
